Mint the note below as a Bettyjane human **pin**: a durable, human-authored
memory signed with the human key, then report the resulting txid.

Run the bj CLI from the Bettyjane project root with the Bash tool:

`bun bin/bj.ts pin "<the note below>"`

(If the current directory is not the Bettyjane repo because the plugin is
installed elsewhere, run the same command against `$CLAUDE_PLUGIN_ROOT/bin/bj.ts`
instead.) The note is written permanently and publicly on chain.

Note to pin: $ARGUMENTS
